🗺️ 어디고? 개인정보처리방침

Privacy Policy · 최종 수정일: 2026년 5월 22일

1. 개요

어디고?는 어르신과 교통약자를 위한 대중교통 길찾기 앱입니다.
본 앱은 서비스 제공에 꼭 필요한 최소한의 정보만 사용하며, 수집된 정보를 외부에 제공하거나 별도로 저장하지 않습니다.

2. 수집하는 정보 및 이용 목적

📍 위치 정보 (GPS)
  • 현재 위치를 파악하여 출발지 기준 대중교통 경로를 안내합니다.
  • 위치 정보는 경로 검색 시에만 일시적으로 사용되며, 서버에 저장되지 않습니다.
  • 앱을 사용하지 않는 동안에는 위치 정보를 수집하지 않습니다.

🎤 마이크 (음성 인식)
  • 음성으로 목적지를 말씀하시면 검색할 수 있도록 마이크를 사용합니다.
  • 음성 인식은 기기 내 브라우저(Chrome)의 Web Speech API를 통해 처리되며, 녹음 파일은 저장되지 않습니다.
  • 마이크 버튼을 누를 때에만 일시적으로 활성화됩니다.
✅ 위치 정보와 음성 정보는 앱 서버에 저장되지 않으며, 제3자에게 제공되지 않습니다.

3. 제3자 서비스 이용

경로 검색을 위해 아래 외부 API를 사용합니다. 각 서비스의 개인정보처리방침을 따릅니다.

  • 카카오 로컬 API — 장소 검색 (카카오 개인정보처리방침 적용)
  • ODsay API — 대중교통 경로 조회 (ODsay 개인정보처리방침 적용)

4. 권한 철회

위치 및 마이크 권한은 언제든지 스마트폰 설정에서 철회할 수 있습니다.
권한을 허용하지 않아도 텍스트 입력 방식으로 앱을 이용할 수 있습니다.

  • 안드로이드: 설정 → 앱 → 어디고? → 권한
